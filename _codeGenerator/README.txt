Используемые CSS:

.detailGrid - таблица в шаблоне view.tpl (при табличном отображении)
.detailGrid .title - ячейки левого столбика этой таблицы (заголовки)
.detailGrid .data - ячейки правого столбика этой таблицы (значения)

.hidden - класс для формы, делающий ее невидимой (в форме лежит один input hidden) (view.tpl)

.dataGrid - таблица в шаблонах list.tpl, admin_list.tpl (при табличном отображении)

.biglabel - жирные лэйблы для страницы edit.tpl (на дивах)
